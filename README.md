# Node-Js-Voting-App

Aplikasi untuk menghitung vote pemilihan secara realtime

# Instalasi
git clone https://github.com/hasan000amin/Node-Js-Voting-App.git

cd Node-Js-Voting-App

npm install

node server.js

# Akses

http://localhost:3000

http://localhost:3000/result
